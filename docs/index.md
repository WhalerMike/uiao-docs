# UIAO Document Compiler v1.0

The UIAO Document Compiler v1.0 is a YAML‑driven, template‑based system for generating leadership‑grade modernization documents from a single canonical content graph.

This site publishes:

- UIAO Leadership Briefing v1.0  
- Program Vision v1.0  
- Unified Architecture v1.0  
- TIC 3.0 Roadmap v1.0  
- Modernization Timeline v1.0  
- FedRAMP 22 Compliance Summary v1.0  
- Zero Trust Narrative v1.0  
- Identity Plane Deep Dive v1.0  
- Telemetry Plane Deep Dive v1.0  

All documents are generated from:

- `canon/uiao_leadership_briefing_v1.0.yaml`  
- `templates/*.md.j2`  

This keeps the canon **single‑sourced**, **drift‑resistant**, and **automation‑ready**.
